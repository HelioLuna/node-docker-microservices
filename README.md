# Microservice Architecture

Protótipo de serviço de uma arquitetura de microsservices em docker.

### Stack

Foi utilizado um ambiente simples construido com NodeJS e Docker for Windows.

- NodeJS v10.0.0
- Docker For Windows v18.03.0-ce
